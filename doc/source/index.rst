ATPPeX documentation
====================

Advanced Tractography-based Parcellation and Pathway Explanation (ATPPeX) is a pipeline for processing diffusion MRI data. It includes various modules for tasks such as registration, ROI analysis, connectivity calculations, and parcellation.
